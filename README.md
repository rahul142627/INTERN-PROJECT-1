Expense Tracker and Budget Manager
📌 Description

This is a GUI-based Expense Tracker and Budget Manager developed using Java Swing. The application helps users manage their daily expenses, categorize spending, and monitor their monthly budget efficiently.

🚀 Features

Add and categorize expenses

View expense records in a table

Delete selected expenses

Search and filter expenses by category

Set monthly budget

Budget limit alert system

Expense analysis (total & highest category)

Data persistence using CSV file

🛠️ Technologies Used

Java

Swing (GUI)

File Handling (CSV)

Collections (ArrayList)

OOP Concepts
